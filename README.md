# sao-hcy-nm

> Personal favorite new project template - SAO generator

## Usage

Install [SAO](https://github.com/saojs/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From npm

```bash
sao hcy-nm my-project
```

### From git

```bash
sao HcySunYang/sao-hcy-nm my-project
```
